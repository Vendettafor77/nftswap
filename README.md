# NFTスワッププラットフォーム

イーサリアムブロックチェーン上でのNFT（非代替性トークン）の取引を可能にするプラットフォームです。

![NFTスワップバナー](./frontend/public/banner.png)

## 📋 プロジェクト概要

NFTスワッププラットフォームは、ユーザーがNFTを簡単に取引できる分散型マーケットプレイスです。主な機能は以下の通りです：

- NFTの閲覧と購入
- 所有NFTの出品（販売）
- VenAPE NFTのミント（新規作成）
- NFTの転送
- 取引履歴の確認

## 🖥️ UI インターフェース

### メインページ

メインページには2つのタブがあります：

**「NFTを閲覧・購入」タブ**

- 出品中のすべてのNFTをグリッド表示
- 検索、フィルタリング、並べ替え機能
- 各NFTカードには画像、名前、コレクション名、価格が表示
- 「購入する」ボタンで直接購入可能

**「NFTを出品する」タブ**

- 所有NFTの選択と出品価格設定
- マーケットへの出品手続き

### マイNFTページ

- 所有しているすべてのNFTを表示
- NFTの検索とフィルタリング
- 各NFTの管理オプション（出品、転送など）
- バッチ選択機能による複数NFTの同時管理

### VenAPEミントページ

- VenAPE NFTの新規作成（ミント）機能
- IDを指定してミント可能
- ミント成功後の確認メッセージ

### 取引履歴ページ

- すべてのNFT取引履歴を時系列で表示
- 購入、販売、ミント、転送などの取引タイプ別表示
- 詳細情報と取引確認リンク

## 📘 使用方法

### ウォレット接続

1. 右上の「ウォレット接続」ボタンをクリック
2. お好みのウォレットプロバイダー（MetaMask、WalletConnectなど）を選択
3. 接続承認プロンプトに従う
4. 接続成功後、アドレスとETH残高が表示されます

### NFTの閲覧と購入

1. メインページの「NFTを閲覧・購入」タブで利用可能なNFTを閲覧
2. 検索バーやフィルターを使って特定のNFTを探す
3. 「購入する」ボタンをクリックして購入手続きを開始
4. ウォレットで取引を確認
5. 取引完了後、NFTはあなたの所有物になります

### NFTの出品

1. メインページの「NFTを出品する」タブに移動
2. 出品したいNFTを選択（「選択する」ボタン）
3. 販売価格（ETH）を設定
4. 「マーケットに出品する」ボタンをクリック
5. 承認取引とリスティング取引を確認
6. 成功すると、NFTがマーケットプレイスに表示されます
